# Mexico-Covid-Analysis
Covid-19 Mexico Data Analysis

An exploratory data analysis project using PostgreSQL to analyze COVID-19 patient data from Mexico, focusing on mortality rates, demographics, and comorbidities.

Data sourced from Kaggle:
https://www.kaggle.com/datasets/meirnizri/covid19-dataset?resource=download

Tools Used:
SQL (PostgreSQL, pgAdmin)
Power BI (planned)
